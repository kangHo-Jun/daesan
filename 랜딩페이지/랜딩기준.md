## 4. 배경 애니메이션 스펙

### 방식
화이트 배경 위 건축 키워드 부유 → 검색창으로 흡입

### 키워드 목록
```
합판, 각재, 석고보드, 단열재, MDF, OSB, 창호, 집성목,
코팅합판, 방음재, 플로링, 흡음재, 문틀, 마루, 방화문,
목재, 보드, 텍스, 단열, 자재
```

### 동작 단계
| 단계 | 시점 | 내용 |
|------|------|------|
| Phase 1 | 0 ~ 1.6s | 키워드 배경에서 둥둥 부유 |
| Phase 2 | 1.6s ~ | 키워드 하나씩 검색창으로 흡입 |
| Phase 2 | 1.6s ~ | 검색창 주변 골드 글로우 활성화 |
| Loop | 상시 | 흡입된 자리에 새 키워드 등장 → 무한 반복 |

### 키워드 스타일
```
font-family : Pretendard, sans-serif
font-weight : 900
font-size   : 11px ~ 18px (랜덤)
color       : rgba(18, 54, 40, 0.055 ~ 0.12)
```

### 흡입 파라미터
```
활성화 딜레이  : 1.6초
전환 확률      : 매 프레임 0.4%
가속도         : dx * 0.014 / dy * 0.014
감속계수       : 0.86
소멸 조건      : 검색창 16px 이내 OR alpha <= 0
글로우 컬러    : rgba(201, 168, 76, 0.13)
글로우 반경    : 90px
```
