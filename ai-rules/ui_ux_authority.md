# UI UX Authority – cafe24_detail_ver10

## Layout Principle
- Enterprise density layout
- Left-right 2 column product detail
- Sticky price panel
- Expandable specification table

## Visual Style
- Glassmorphism (light blur, 8–16px radius)
- Navy primary (#002244)
- Accent blue (#0055ff)
- Soft gray background (#f8f9fa)

## UX Rules
- Variant grouping must not scroll jump
- Parent-child product hierarchy must be visually clear
- No long vertical overflow explosion
- Price hover = graph preview allowed
- Scroll reveal animation allowed

## Typography
- Strong hierarchy (H1 bold)
- Data values emphasized
- Dense but readable spacing
