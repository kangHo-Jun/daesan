# 요구사항 정의서 (PRD.md)

## 프로젝트 개요
Google Sheet 데이터를 기반으로 Cafe24 상품상세(`<div id="prdDetail">`) 영역에 삽입 가능한 **단일 HTML 템플릿**을 자동 생성하는 시스템

## 핵심 타겟
- **1차**: 자트 내부 운영자 (제품 등록 담당자)
- **2차**: 건설자재 쇼핑몰 방문 고객 (B2B 리모델링업자)

## 주요 기능 (MVP Scope)

### 필수 기능 1: 4개 섹션 템플릿 생성

#### 섹션 1. 회사 차별점 (4개 통계 카드)
```
┌────────────────────────────────────────────────────────┐
│  📊 40년차+     🏢 1702개+     💰 467억+     💡 12명+   │
│  사업연차      관리거래처      거래금액      정규직원수   │
└────────────────────────────────────────────────────────┘
```
- 아이콘(SVG) + 숫자 + 라벨 + "+" 애니메이션
- 제품별 다른 값 (Sheet 연동)
- 4개 카드 균등 배치
- 흰색 배경, 미니멀 디자인

#### 섹션 2. 가격 동향 (현재가 + 변동율)
```
┌─────────────────────────────────┐
│  현재가: ₩18,500                │
│  ▼ 3.6%  (₩700)                │
│  전주 대비 하락                  │
│  [hover 시 7일 그래프 팝업]      │
└─────────────────────────────────┘
```
- 일주일 전 대비 상승/하락/보합 표시
- 상승: 빨강(#ef5350) + ▲
- 하락: 파랑(#2196F3) + ▼
- 보합: 회색(#757575) + —
- Hover 시 7일 그래프 팝업 (Satori 이미지)

#### 섹션 3. 만화 4컷 (2×2 그리드)
```
┌───────────┬───────────┐
│           │           │
│  [홀더1]  │  [홀더2]  │
│           │           │
└───────────┴───────────┘
┌───────────┬───────────┐
│           │           │
│  [홀더3]  │  [홀더4]  │
│           │           │
└───────────┴───────────┘
```
- 이미지 홀더 + URL만 교체
- 반응형 레이아웃 (모바일: 1열)
- 각 홀더 약 370px

#### 섹션 4. 제품 상세 (좌우 2단)
```
┌───────────┬─────────────────┐
│           │  📌 용도         │
│           │  • 용도1         │
│  [이미지] │  • 용도2         │
│  홀더     │                  │
│           │  🏗️ 사용처       │
│           │  • 사용처1       │
│           │  • 사용처2       │
│           │                  │
│           │  ✨ 특징          │
│           │  • 특징1         │
│           │  • 특징2         │
└───────────┴─────────────────┘
```
- 왼쪽: 제품 이미지
- 오른쪽: 마크다운 → HTML 변환 콘텐츠
- 마크다운 지원: 제목, 리스트, 볼드, 링크

---

### 필수 기능 2: Google Sheet 데이터 매핑

Sheet 컬럼 → HTML placeholder 자동 치환
```
예: {{stat1_number}} → 40
    {{current_price}} → 18,500
    {{detail_content_md}} → HTML 변환
```

---

### 필수 기능 3: Cafe24 최적화

- **790px 고정폭** (모바일: 좌우 16px 여백)
- **zart- prefix 네임스페이스** (ID/클래스 충돌 방지)
- **단일 HTML 파일** (`<style>` + `<div>` + `<script>`)
- **IntersectionObserver 스크롤 애니메이션**
  - 각 섹션이 뷰포트 진입 시 fade-up
  - rootMargin: '-80px'
  - threshold: 0.15

## 사용자 흐름

1. **운영자**: Google Sheet에 제품 데이터 입력
2. **시스템**: HTML 파일 자동 생성 (`product_1164.html`)
3. **운영자**: Cafe24 에디터에 복사-붙여넣기
4. **고객**: 상품상세 페이지에서 4개 섹션 확인
   - 스크롤 시 섹션별 등장 애니메이션
   - 가격에 hover 시 그래프 팝업

## UI/UX 요구사항

### 색상 시스템
```css
/* 브랜드 색상 */
--zart-primary: #2d5a3d;      /* 녹색 */
--zart-price-up: #ef5350;     /* 빨강 */
--zart-price-down: #2196F3;   /* 파랑 */
--zart-price-flat: #757575;   /* 회색 */

/* 중립 색상 */
--zart-text: #333333;
--zart-text-sub: #666666;
--zart-border: #e0e0e0;
--zart-bg: #f8f9fa;
```

### 타이포그래피
```css
--zart-font-base: 15px / 400;
--zart-font-title: 20px / 700;
--zart-font-price: 28px / 700;
--zart-font-sub: 13px / 400;
```

### 애니메이션
```css
/* 스크롤 등장 */
transition: opacity 0.6s cubic-bezier(0.2, 0.8, 0.2, 1),
            transform 0.6s cubic-bezier(0.2, 0.8, 0.2, 1);

/* Hover 그래프 팝업 */
transition: opacity 0.18s ease,
            transform 0.18s ease;
```
