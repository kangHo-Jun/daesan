# Cafe24 상세페이지 다이나믹 기법 요약 (cafe24_ver1.html)

`cafe24_ver1.html` 파일에 적용된 주요 인터랙션 및 고도화된 다이나믹 기법은 다음과 같습니다.

### 1. 스크롤 기반 인터랙티브 애니메이션
- **기술**: `Intersection Observer API` & CSS3 Transitions
- **효과**: 사용자가 화면을 내릴 때 각 섹션과 요소들이 **순차적으로 부드럽게 등장**합니다.
  - `zart-scroll-reveal`: 하단에서 위로 페이드인
  - `zart-slide-left/right`: 좌우에서 슬라이드 인
  - `zart-scale-up`: 중앙에서 커지며 강조

### 2. 실시간 데이터 스트리밍 시뮬레이션
- **카운터 애니메이션**: 견적 요청 건수가 0부터 시작하여 실제 숫자까지 올라가는 애니메이션을 통해 **활발한 거래 신뢰도**를 시각화합니다.
- **업데이트 시간 로직**: 페이지 로드 시점을 기준으로 마지막 업데이트 시간을 표시하여 **최신 정보임을 강조**합니다.

### 3. 인터랙티브 데이터 시각화 (SVG 차트)
- **차트 확장**: '가격 동향' 섹션에 마우스를 올리면 숨겨져 있던 **SVG 가격 그래프**가 `max-height` 애니메이션과 함께 펼쳐집니다.
- **SVG 폴리라인**: 복잡한 드로잉 없이 SVG `polyline`을 사용하여 깔끔한 가격 등락 곡선을 구현했습니다.
- **호버 툴팁**: 각 데이터 포인트에 마우스를 올리면 상세 견적 숫자를 확인할 수 있는 툴팁 기능을 내장했습니다.

### 4. 사용자 경험(UX) 마이크로 인터랙션
- **스펙 팝업**: 제품 이미지 영역에 호버 시 상세 스펙이 반투명 블러(`backdrop-filter`) 레이어로 나타나 정보 탐색을 돕습니다.
- **시각적 계층**: 섹션 제목의 언더라인 애니메이션 등을 통해 현재 읽고 있는 위치를 명확히 인지하게 합니다.

### 5. 24시간 프로세스 타임라인 프로그레스
- **스크롤 동기화**: 사용자의 스크롤 위치에 따라 타임라인의 **게이지 바(Progress Bar)**가 채워집니다.
- **노드 활성화**: 해당 공정 단계에 도달하면 원형 노드가 강조되며 시인성을 높입니다.

---
> [!NOTE]
> 모든 코드는 카페24의 WAF(보안필터)를 우회할 수 있도록 인라인 스타일과 IIFE(즉시실행함수) 구조로 최적화되었습니다.
