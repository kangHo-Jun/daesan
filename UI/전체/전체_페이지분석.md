## 3. 문제 진단 (6건)

### [문제1] 색상 시스템 Authority 위반 ★★★

- **유형:** 시각 불일치
- **원인:** CSS `--zart-primary: #2d5a3d` (초록 계열)
  Authority 지정: Navy `#002244` + Accent `#0055ff`
- **영향:** 브랜드 색상 정체성 붕괴, 카페24 메인과 충돌
- **수정방향:** CSS 변수 전면 교체

```css
/* 교체 전 */
--zart-primary: #2d5a3d;
--zart-primary-light: #3d6d4d;

/* 교체 후 */
--zart-primary: #002244;
--zart-accent: #0055ff;
```

---

### [문제2] Section 3 구조 미완성 ★★★ (최우선)

- **유형:** 구조 공백
- **원인:** HTML 63번째 줄 이후 Section 3이 `<img>` 1개로만 구성됨
  CSS에 `.zart-gemini-infographic`, `.zart-g-header`, `.zart-g-mid`, `.zart-g-foot` 등 **완전한 컴포넌트 시스템이 정의되어 있으나 HTML에서 미사용**
- **영향:** `/section3.png` 파일 없으면 핵심 섹션 전체 공백
- **수정방향:** CSS 기정의 컴포넌트로 HTML 구현

```
.zart-gemini-infographic
  ├── .zart-g-header     (제품이미지 + 제품명 + E1뱃지)
  ├── .zart-g-mid        (비교카드 + 도넛게이지)
  └── .zart-g-foot       (용도 가이드 3카드)
```

---

### [문제3] 설득 구조 역순 + CTA 전무 ★★★

- **유형:** 흐름 설계
- **원인:** 제품 소개 없이 통계/가격 먼저 노출. 페이지 전체에 CTA 0개
- **영향:** 컨텍스트 없는 숫자 → 신뢰 형성 불가, 구매 전환 불가
- **수정방향:** 섹션 순서 재배치 + CTA 블록 추가

### [문제5] Section 5 카드 색상 난립 ★★

- **유형:** 카드 시스템 불일치
- **원인:** 4개 카드 배경 `#e8f5e9`, `#f1f8e9`, `#dcedc8`, `#c5e1a5` 인라인 하드코딩
- **영향:** 카드 통일성 파괴, Authority 팔레트 이탈, Glassmorphism 미적용
- **수정방향:** 전 카드 `white` + Navy accent border 통일

---

### [문제6] 컨테이너 폭 불일치 ★

- **유형:** 구조 정렬

- **영향:** Section 3이 타 섹션보다 약간 넓어 정렬 어긋남
- **수정방향:** 800px 단일 기준 통일
