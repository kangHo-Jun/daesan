™## **1\. 프로젝트 목표**

* 전자상거래용 고품질 제품 상세 인포그래픽 생성 (리스케일 v3)
* **한글 깨짐 없이 완벽한 렌더링** 필수  
* 입력된 JSON 데이터에 기반한 정확한 정보 시각화  
* 전문적이고 깔끔한 디자인, 미세한 그림자 효과로 깊이감 부여

## **2\. 캔버스 및 스타일 정의**

* **캔버스 크기:** 800 x 1200 px (웹 UI 최적화 리스케일 버전)
* **전체 배경:** #FFFFFF (Pure White)
* **폰트 설정 (기본 대비 20% 축소):**
  * NotoSansKR-Bold.ttf: 주요 타이틀(24-32px), 강조 데이터(수치, 18-22px), 뱃지 메인 텍스트
  * NotoSansKR-Regular.ttf: 서브 타이틀(14-16px), 일반 라벨(12-14px), 본문 텍스트
  * ⚠️ **주의:** 반드시 지정된 커스텀 폰트만 사용하여 서버 렌더링 시 한글 깨짐을 방지할 것.
* **레이아웃 원칙:** 섹션 간의 **수직 간격(Vertical Margin)을 기존 대비 20% 이상 압축**하여 웹 브라우저에서의 가독성과 밀도를 극대화함.

## **3\. 레이아웃 구조 (3단 구성)**

┌────────────────────────────────────────────────────────┐  
│ 캔버스 전체 (800x1200, #FFFFFF)                        │  
│ ┌─ SECTION 1: 제품 메인 (배경 #F5F0E8) ──────────────┐ │  
│ │ [좌측: 대형 제품 이미지] | [우측: 제품명/스펙/뱃지]  │ │  
│ └────────────────────────────────────────────────────┘ │  
│ ┌─ SECTION 2: 특성 비주얼 (배경 #F0EDE6) ────────────┐ │  
│ │ 타이틀 영역                                        │ │  
│ │ [비교 카드(좌)] [비교 카드(우)] | [도넛 차트]      │ │  
│ └────────────────────────────────────────────────────┘ │  
│ ┌─ SECTION 3: 용도 가이드 (배경 #F2F4F5) ────────────┐ │  
│ │ 타이틀 영역 (Usage Guide)                          │ │  
│ │ [카드1(컬러아이콘)] [카드2] [카드3] (배경 #FFFFFF)     │ │  
│ └────────────────────────────────────────────────────┘ │  
└────────────────────────────────────────────────────────┘

## **4\. 섹션별 상세 렌더링 가이드**

**SECTION 1: 제품 메인**

* **배경색:** data.section\_1.bg\_color (\#F5F0E8) 적용  
* **제품 이미지:** product\_image\_url의 이미지를 좌측에 배치하고 부드러운 그림자 효과 부여  
* **텍스트:** 우측 상단에 product\_name과 product\_density를 배치  
* **뱃지:** 우측 하단에 grade\_badge 정보(E1, 친환경 인증)를 포함한 원형 배지 생성

**SECTION 2: 특성 비주얼**

* **배경색:** data.section\_2.bg\_color (\#F0EDE6) 적용  
* **비교 영역:** 일반 MDF와 고밀도 HDF의 단면 사진 및 라벨을 좌측에 나란히 배치  
* **차트 영역:** 우측에 '휨 강도'를 나타내는 입체적인 도넛 차트 렌더링 및 center\_value(35 N/mm²) 표시

**SECTION 3: 용도 가이드**

* **배경색:** data.section_3.bg_color (#F2F4F5) 적용  
* **카드 스타일:** 각 카드의 배경색은 **#FFFFFF**로 설정하고 테두리를 부드럽게 라운딩 처리하여 입체감을 부여함
* **아이콘 스타일:** 단순 라인 아이콘이 아닌, **제품의 톤과 어울리는 컬러(브라운, 우드톤 등)가 포함된 채우기 스타일의 아이콘**을 사용할 것
* **구성:** 가구재, 마루판, 건축 내장재를 상징하는 아이콘과 라벨이 포함된 3개의 카드를 가로로 배치

## **6\. 시스템 명령 (System Prompt)**

위의 구조와 데이터 JSON을 분석하여, 정의된 스타일 가이드를 준수하는 최종 인포그래픽 이미지를 생성하라. 모든 텍스트와 시각 요소는 데이터와 일치해야 하며, 전문적인 이커머스 디자인 품질을 유지해야 한다.
