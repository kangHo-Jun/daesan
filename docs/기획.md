# 상품정보 섹션 템플릿 기획

> **목적**: DB 연동을 위한 재사용 가능한 템플릿
> **적용 환경**: 카페24 `prdDetail` 영역
> **고정폭**: 790px
> **작성일**: 2025-02-03

---

## 🎯 핵심 원칙

1. 모든 내용은 **DB에서 받아옴** → 템플릿은 구조만
2. 실제 데이터·이미지는 **placeholder로 표시**
3. DB 연동 포인트는 주석으로 명확히 표시
4. 복잡함 제거 → 간결하고 깔끔하게

---

## 📐 전체 구조

```
┌─────────────────────────────────┐  790px
│  1. 회사 차별점 (한줄 배너)      │
├─────────────────────────────────┤
│  2. 가격 동향 (일주일 상승/하락) │
├─────────────────────────────────┤
│  3. 만화 4컷 (2x2 홀더)         │
├─────────────────────────────────┤
│  4. 제품 상세 (좌우 2단)         │
└─────────────────────────────────┘
```

---

## 섹션 1 | 회사 차별점

### 구조
```
┌─────────────────────────────────┐
│  {{항목1}}  |  {{항목2}}  |  {{항목3}}  │
└─────────────────────────────────┘
```

### 규칙
- 녹색 배경 (#2d5a3d) + 흰색 텍스트
- 한줄로 " | " 구분
- 텍스트는 DB에서 받아옴

### DB 연동 포인트
```
항목1, 항목2, 항목3
```

---

## 섹션 2 | 가격 동향

### 구조
```
┌─────────────────────────────────┐
│  현재가: ₩{{현재가}}             │
│  ▼ {{변동율}}%  (₩{{변동금액}}) │
│  전주 대비 하락                  │
└─────────────────────────────────┘
```

### 규칙
- 상승 시: 빨강 (#ef5350) + ▲
- 하락 시: 파랑 (#2196F3) + ▼
- 보합 시: 회색 (#757575) + —
- 일주일 단위로 가격 변동만 표시
- 차트 없음, 재고/출고/업데이트 없음

### DB 연동 포인트
```
현재가, 일주일 전 가격 → 변동율·변동금액은 클라이언트에서 계산
```

## 섹션 3 | 만화 4컷

### 구조
```
┌───────────┬───────────┐
│           │           │
│  [홀더1]  │  [홀더2]  │  각 ~370px
│           │           │
└───────────┴───────────┘
┌───────────┬───────────┐
│           │           │
│  [홀더3]  │  [홀더4]  │
│           │           │
└───────────┴───────────┘
```

### 규칙
- 레이아웃: 2열 × 2행
- 이미지는 사용자가 직접 제공
- 템플릿은 홀더 박스만 제작
- 각 홀더에 이미지 경로만 교체하면 됨

### DB 연동 포인트
```
홀더1~4 각각의 이미지 경로 (src)
```

## 섹션 4 | 제품 상세

### 구조
```
┌───────────┬─────────────────┐
│           │  📌 용도         │
│           │  • {{용도1}}     │
│  [이미지] │  • {{용도2}}     │  좌우 2단
│  홀더     │                  │
│           │  🏗️ 사용처       │
│           │  • {{사용처1}}   │
│           │  • {{사용처2}}   │
│           │                  │
│           │  ✨ 특징          │
│           │  • {{특징1}}     │
│           │  • {{특징2}}     │
└───────────┴─────────────────┘
```

### 규칙
- 왼쪽: 제품 이미지 홀더 (1장)
- 오른쪽: 용도 / 사용처 / 특징 (각각 리스트)
- 모든 텍스트는 DB에서 받아옴

### DB 연동 포인트
```
제품 이미지 경로, 용도 리스트, 사용처 리스트, 특징 리스트
```
