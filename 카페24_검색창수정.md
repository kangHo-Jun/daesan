# Cafe24 검색창 스타일 수정 가이드

> 작성일: 2026-03-30
> 난이도: 초보자 가능
> 적용 파일: `custom/main/main_visual.html`

---

## 배경

Cafe24 기본 검색창은 여러 CSS 파일이 겹쳐 적용되어 스타일 수정이 까다롭습니다.
이 문서는 검색창을 원하는 디자인으로 바꾸는 방법을 단계별로 정리합니다.

---

## 문제점 & 해결 과정

### 문제 1 — 검색창 테두리가 다크그린으로 고정됨
**원인**: `input#keyword` 요소에 기본 테두리가 적용되어 있음
**해결**: `input` 에 `border: none !important` 추가

### 문제 2 — fieldset 바깥에 이중 테두리가 보임
**원인**: 브라우저 기본 `fieldset` 스타일 + `outline-color` 전역 설정
**해결**: `outline: none !important`, `appearance: none !important` 추가

### 문제 3 — 돋보기 아이콘 색상 변경 안 됨
**원인**: PNG 이미지라 color CSS 적용 불가, CSS `!important` 가 inline style 차단
**해결**: `filter` 속성으로 색상 변환

---

## 최종 적용 CSS

`main_visual.html` 의 `<style>` 태그 안에 아래 코드 추가:

```css
/* ── input 테두리 완전 제거 ── */
.main_visual_section #keyword,
.main_visual_section .inputTypeText {
  border: none !important;
  outline: none !important;
  box-shadow: none !important;
  background: transparent !important;
}

/* ── fieldset 스타일 ── */
.main_visual_section fieldset {
  border: 1.5px solid rgba(201,168,76,0.55) !important;  /* 골드 테두리 */
  border-radius: 40px !important;
  box-shadow: 0 2px 20px rgba(18,54,40,0.07) !important;
  background: #fff !important;
  outline: none !important;
  appearance: none !important;
  -webkit-appearance: none !important;
}

/* ── 검색 버튼 — 배경 투명, 아이콘만 골드 ── */
.main_visual_section .search_btn {
  background: transparent !important;
  border: none !important;
  box-shadow: none !important;
  border-radius: 0 !important;
  transition: transform .2s;
}
.main_visual_section .search_btn:hover {
  transform: scale(1.08);
}

/* ── 돋보기 아이콘 골드 (#C9A84C) ── */
.main_visual_section .search_btn img {
  filter: brightness(0) saturate(100%)
          invert(67%) sepia(43%) saturate(600%)
          hue-rotate(358deg) brightness(98%) contrast(88%) !important;
  width: 24px !important;
  height: 24px !important;
}
```

---

## 핵심 원리

| 문제 | 해결 방법 |
|------|----------|
| CSS가 안 먹힐 때 | `!important` 추가 |
| PNG 아이콘 색상 변경 | `filter` 속성 사용 |
| 브라우저 기본 스타일 제거 | `appearance: none` |
| 어디서 스타일이 오는지 모를 때 | F12 Console에서 `getComputedStyle()` 로 확인 |

## 디버깅 방법 (F12 Console)

```javascript
/* 특정 요소의 스타일 확인 */
const el = document.querySelector('#vanta-birds-bg fieldset')
console.log('border:', getComputedStyle(el).border)
console.log('outline:', getComputedStyle(el).outline)

/* 어느 CSS 파일에서 오는지 확인 */
for(let s of document.styleSheets){
  try{
    for(let r of s.cssRules){
      if(r.selectorText && r.selectorText.includes('fieldset')){
        console.log(s.href, '|', r.selectorText, '|', r.style.cssText)
      }
    }
  }catch(e){}
}

/* 부모 요소 순서대로 스타일 확인 */
let el2 = document.querySelector('#vanta-birds-bg fieldset')
for(let i=0; i<5; i++){
  el2 = el2.parentElement
  const cs = getComputedStyle(el2)
  console.log(i, el2.tagName, el2.className, '| border:', cs.border)
}
```

## 주의사항

- `!important` 없이는 기존 CSS에 덮어씌워짐
- PNG 이미지는 `color` 속성으로 색상 변경 불가 → 반드시 `filter` 사용
- Cafe24 `searchHeader.css` 모듈은 직접 수정 금지
- `fieldset` 의 `appearance: none` 은 브라우저 기본 테두리 제거에 필수
