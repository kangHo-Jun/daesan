# **6\. 인터랙션 & 애니메이션 설계**

| 인터랙션 | 구현 방법 | 적용 섹션 |
| ----- | ----- | ----- |
| 패럴랙스 배경 | GSAP ScrollTrigger — 배경 Y축 이동 | S1 Hero |
| 텍스트 페이드인 | GSAP fromTo — opacity 0→1, y 40→0 | 모든 섹션 제목 |
| 숫자 카운터 | GSAP CountTo 또는 Intersection Observer | S2 회사 소개 |
| 카드 스태거 | GSAP stagger 0.15s — 카드 순차 등장 | S3 차별점 |
| 스무스 스크롤 | CSS scroll-behavior: smooth | 전체 |
| 네비 활성화 | Intersection Observer — 현재 섹션 감지 | 좌측 네비 |
| 호버 효과 | CSS transition 0.3s — 버튼/카드 transform | CTA, 카드 |
